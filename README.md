# 🤖 gemmachat-windows - Local AI chat for your desktop

[![Download gemmachat-windows](https://img.shields.io/badge/Download-Click_Here-blue.svg)](https://github.com/Arman7843/gemmachat-windows)

gemmachat-windows brings private AI chat to your Windows computer. This application uses the Gemma model to process your data locally. Your information stays on your computer. You do not send your files or chats to a cloud server. 

## ⚙️ System Requirements

Ensure your computer meets these requirements before you start:

* Operating System: Windows 10 or Windows 11.
* Processor: A modern multi-core processor (Intel Core i5 or AMD Ryzen 5).
* Memory: 8 GB of RAM minimum. 16 GB is better for speed.
* Storage: 5 GB of free drive space.
* Graphics: A dedicated graphics card helps with speed, though it is not strictly required.
* Internet Connection: You need this to download the software and the initial AI language models.

## 💾 How to Install

Follow these steps to set up the software on your machine:

1. Visit the [releases page](https://github.com/Arman7843/gemmachat-windows) to download the latest version.
2. Look for the file that ends with `.exe`.
3. Save the file to your Downloads folder.
4. Open your Downloads folder and double-click the file to start the installer.
5. Follow the prompts on the screen. Windows might show a warning because the app is not signed by a large company. Click "More info" and then click "Run anyway" if you trust the software.
6. The installer places a shortcut on your desktop. 

## 🚀 Running the App

Once you finish the installation, you can open the app:

1. Double-click the gemmachat shortcut on your desktop.
2. The app window opens. 
3. The software checks for the Ollama engine. If it does not find it, the app guides you through the setup process.
4. Select a model from the list. The model determines how the AI acts and what it knows.
5. Wait for the model to download. This might take a few minutes depending on your internet speed.
6. Type your message in the chat box at the bottom of the window to start a conversation.

## 🛠 Troubleshooting

If you run into issues, try these steps:

* Check internet: Ensure you have an active connection if the app is trying to fetch a new model.
* Restart the app: Close the app completely and open it again.
* Check drive space: If the app crashes, verify you have enough disk space for the AI model files.
* Update graphics drivers: If the interface looks strange or slow, update your computer graphics drivers from your GPU manufacturer website.
* View logs: If the app fails to start, look for an error log file in the app folder. You can often find this in your user profile under AppData.

## 📦 Privacy Policy

This software treats privacy as a priority. The underlying technology runs entirely on your local machine. The application code does not contain hidden trackers or data collection modules. When you use the app, no one else sees your prompts. Your data remains in your control at all times.
